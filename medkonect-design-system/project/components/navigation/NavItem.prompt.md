**NavItem** — sidebar navigation row.

```jsx
<NavItem icon="dashboard" href="#" active>Overview</NavItem>
<NavItem icon="calendar_today" href="#">Appointments</NavItem>
<NavItem icon="folder_shared" href="#">Medical Records</NavItem>
<NavItem icon="videocam" href="#">Teleconsult</NavItem>
```

Active rows take a tonal-teal fill with a filled icon. Use one active item per rail.
